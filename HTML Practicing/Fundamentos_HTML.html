H1 - h6 são elementos para títulos

p é um elemento para adicionar parágrafos: blocos de textos e imagens

    ol é um elemento para listas ordenadas como listas númericas por exemplo.
  1. cada item da lista é um "li" (list item)
    ul é um elemento para listas NÃO-ordenadas como bullet points
  • cada item da lista é um "li" (list item)
a é um elemento para gerar hyperlinks (anchor tag); Toda anchor tag precisa ter um "atribute" que é uma referência que vai dentro da tag; Nesse caso indica o endereço para onde aquele link direciona ao ser clicado

A imagem (img) é um elemento que não tem abertura e fechamento, assim como a anchor tag, ela recebe os atributos direto dentro da tag; são eles "src" que é "source" o endereço da imagem e "alt" que é a descrição da imagem para acessibilidade
