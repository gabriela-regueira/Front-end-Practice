a é um elemento para gerar hyperlinks (anchor tag); Toda anchor tag precisa ter um "atribute" que é uma referência que vai dentro da tag; Nesse caso indica o endereço para onde aquele link direciona ao ser clicado

A imagem (img) é um elemento que não tem abertura e fechamento, ela recebe os atributos direto dentro da tag; são eles "src" que é "source" o endereço da imagem e "alt" que é a descrição da imagem para acessibilidade

Div é um elemento do tipo "block" que serve para agrupar conteúdos; é um conteiner genérico
Span faz o mesmo que o div, só que é do tipo "inline"; É legal usar esse conteineres para agrupar conteúdos e colocar estilos CSS neles; O span se usa geralmente em trechos de textos
Section tem a mesma função que um Div, só que também tem uma função "semântica", ou seja, ajuda a seccionar partes do conteúdo, usando o termo correto, facilitando a leitura de um leitor de telas por exemplo. Assim como o div e span, você pode adicionar estilo
